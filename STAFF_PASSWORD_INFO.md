## How to Change Your Staff Password

### Step-by-Step Guide:

1. **Login to Staff Panel**
   - Go to Staff Panel login page
   - Enter your email (e.g., `admin@dates.care`)
   - Enter your current password (e.g., `AdminPass2025!`)

2. **Navigate to Staff Members Section**
   - Scroll down to the "Staff Members" card
   - Find the password management section

3. **Fill in Password Change Form**
   - **Target Staff ID:** Enter the email address of the account you want to change (e.g., `admin@dates.care`)
   - **New Password:** Enter your desired new password (min 8 characters)
   - **Confirm New Password:** Re-enter the same new password
   - **Your Manager Password:** Enter your CURRENT password (`AdminPass2025!`) to authenticate

4. **Click "Change Password"**
   - The system will verify your current password
   - If successful, the new password will be set
   - You'll see a success message

### Common Errors:

#### "Manager authentication failed"
**Cause:** You entered the wrong current password in the "Your Manager Password" field.

**Solution:**
- Make sure you're entering your CURRENT password (not your new password)
- For admin@dates.care, the default is: `AdminPass2025!`
- Check for typos, spaces, and correct capitalization
- Password is case-sensitive: capital A, capital P, exclamation mark at the end

#### "New passwords do not match"
**Cause:** The "New Password" and "Confirm New Password" fields don't match.

**Solution:** Make sure both fields have the exact same password.

#### "Password must be at least 8 characters"
**Cause:** Your new password is too short.

**Solution:** Choose a password with at least 8 characters.

## Important Notes:

1. **Current vs New Password:**
   - "Your Manager Password" = Your CURRENT password (used to authenticate)
   - "New Password" = The password you WANT to use going forward

2. **Password Requirements:**
   - Minimum 8 characters
   - Can include letters, numbers, and special characters
   - Case-sensitive

3. **First Time Users:**
   - Use the default passwords listed above
   - Change your password immediately after first login for security

4. **If You Forgot Your Password:**
   - Use the "Reset to Default" button
   - Enter your manager password (if you're a Super User/Admin)
   - The system will generate a new default password

---

## Security Best Practices:

- Change default passwords immediately
- Use strong passwords with mixed characters
- Don't share your password
- Change passwords regularly
- Never write passwords in plain text
